How to Apply CSS to HTML

To apply external CSS to your HTML file, you need to link to a CSS file in the <head> section of your HTML file using the <link> tag. For example: 

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, you can use the <script> tag at the end of your HTML body: 

<script src="script.js"></script>
Alternatively, you can write inline JavaScript within the HTML file using: 
<script>
    alert('Hello, World!');
</script>
